# 🎮 坦克大战 - Battle City

经典坦克大战游戏的HTML/Canvas实现。直接浏览器打开即可游玩！

## 🕹️ 操作方式

| 按键 | 功能 |
|------|------|
| `W` / `↑` | 向上移动 |
| `S` / `↓` | 向下移动 |
| `A` / `←` | 向左移动 |
| `D` / `→` | 向右移动 |
| `空格` | 开火 |
| `P` | 暂停/继续 |

## 🎯 游戏目标

- 消灭所有敌方坦克（每关20辆）
- 保护基地（大本营）不被摧毁
- 每消灭一个敌人 +100分
- 拥有3条生命，全部消耗则游戏结束

## 🏗️ 地图元素

- 🧱 **砖墙** — 可被子弹摧毁
- ⬜ **钢墙** — 坚不可摧
- 🏠 **基地** — 需要守护的目标

## 🔧 运行方式

直接用浏览器打开 `index.html` 即可开始游戏，无需任何依赖。

## 📦 技术栈

- 纯 HTML5 + CSS3 + JavaScript
- Canvas 2D 渲染
- 像素风格画面
